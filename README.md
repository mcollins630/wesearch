# Wesearch

Wesearch is a semantic search library developed by WESO (http://www.weso.es)

## Building Wesearch

To build Wesearch you need a JDK 1.6 and Maven 2.2.1 installed. 

## Building Wesearch

In the Wesearch source directory type

    $ mvn install
